# 기본인자

C++에는 기본인자라는 개념이 있다. 

```c++

#include <iostream>

int add(int a, int b=10) {
	return a + b;
}

int main()
{
	int res = add(5, 7);
	std::cout << "res : " << res << std::endl;

	res = add(5);
	std::cout << "res : " << res << std::endl;

	return 0;
}

```

```
res: 12
res: 15
```

`add`는 인자를 2개 받는 함수지만, b에 기본인자로 10을 지정해줬기 떄문에 인자를 하나만 입력해줘도 작동한다.

코틀린에서도 이와 같이 기본인자를 지정할 수 있다.

```kotlin

fun add(a: Int, b: Int = 10): Int {
	return a + b;
}

fun main() {
	var res = add(5, 7);
	println("res : $res");

	res = add(5);
	println("res : $res");

}

```
